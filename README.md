# Nexus Social Media App 

A full-stack mini social media application built with Node.js, Express, MongoDB, and vanilla JavaScript. This project includes user authentication, posting, profiles, following, searching, and direct messaging.

## Features 

* **User Authentication:** Secure sign-up, login, and password reset functionality using JWT.
* **Posts:** Create text and image posts, edit/delete your own posts.
* **Feed:** View posts from users you follow.
* **Interactions:** Like/unlike posts, comment/delete comments.
* **User Profiles:** View user profiles, see their posts, follower/following counts.
* **Follow System:** Follow and unfollow other users.
* **User Search:** Find users by username.
* **Direct Messaging:** Private conversations between users.
